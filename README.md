# ☕ JAVA SOURCE CODE STORAGE

Đây là nơi lưu trữ các file Java.

---

### 📂 Cấu trúc phân cấp
* **Project:** Các thư mục dự án Java chính.
* **Scripts:** File lẻ và các đoạn mã thực thi nhanh.
* **Docs:** Tài liệu hướng dẫn hoặc ghi chú thuật toán.

### 📝 Quy tắc lưu trữ
* Phân chia theo từng ô riêng biệt để không bị cộng dồn file.
* Mỗi dự án lớn nên nằm trong một thư mục con riêng.
* Xóa sạch file cũ hoặc nén lại trước khi nạp phiên bản mới.
